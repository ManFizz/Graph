# Graph Algorithms

Реализация алгоритма **Беллмана — Форда** для поиска кратчайшего пути в ориентированном взвешенном графе (с возможными отрицательными весами).

## О проекте

Лабораторная работа по дисциплине «Фундаментальные структуры данных и алгоритмы».

Программа читает граф из текстового файла, принимает две вершины и находит кратчайший путь между ними с выводом веса пути и самого маршрута.

## Возможности

- Поиск кратчайшего пути алгоритмом Беллмана-Форда
- Обработка отрицательных весов рёбер
- Детекция отрицательных циклов
- Проверка корректности входных данных

## Технологии

- C++
- Работа с файлами

## Как запустить

1. Скомпилировать программу:
   ```bash
   g++ main.cpp -o bellman_ford
   ```
2. Запустить
   ```bash
   ./bellman_ford
   ```
